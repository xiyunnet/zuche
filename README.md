# 羲云租车系统 微信小程序 （原生）

#### 系统介绍
羲云租车系统，是一款可以部署于同城，或者全国性的租车应用，界面非常的简洁美观，功能全面。前后端均为原生开发，适合功能性二次开发。

#### 应用场景
1. 同城部署，为同城租车公司或者地区公司服务。
2. 全国性部署，为租车公司进行加盟，租车进行收费。
3. 个人部署，为个人租车公司部署服务，提高知名度。
4. 接入出租车数据API。
5. 可以使用为同城物业，保洁等应用。

#### 应用功能说明（排序不按次序）
1. 首页为自建地图控件，产用腾讯地图为基础，界面美观，功能有：展示门店，选点，导航，查询等集一体（可见我们开源地图项目）。
2. 按门店进行展示出租车辆，单页面展示所有出租车，用户选择后，直接展示出租信息，非页面跳转。选择租用时间及展示租金。
3. 租用并支付后，推送订单至商户。
4. 商户端管理全部在手机端进行，如需PC端管理，可开发二维码登录。
5. 商户可以开设多家门店，并对各门店进行管理。
6. 商户可以设置员工，并赋予一定的权限。
7. PC端主要对商户进行审核及管理。
8. 租用车辆，可以在商户的不同门店进行还车（可以开通此功能）。
9. 商户可以设置阶段价租车，提高租车金额。
10. 管理员设置汽车库，商户可以快速导入汽车数据，方便汽车的录入。
11. 首页按用户位置进行由近及远进行搜索的门店排序。
12. 更多功能开发测试中，请以测试内容为准。
